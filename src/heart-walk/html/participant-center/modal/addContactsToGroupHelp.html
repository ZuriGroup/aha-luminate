Add Contacts to Group

You can create your own groups of contacts, for easier email handling.

To add contacts to a group, follow these directions.

  1. Select all the contacts you want to add to a new or existing group.
  2. Click the Add Contacts to Group button.
  3. Either enter a name for a new group, or choose a previously created group form the drop down menu.
  4. Click the Add Contacts to Group button.
